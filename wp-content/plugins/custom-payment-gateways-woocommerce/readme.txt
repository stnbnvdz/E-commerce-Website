Custom payment gateways for WooCommerce.

== Description ==

**Custom Payment Gateways for WooCommerce** plugin lets you add custom payment gateways to WooCommerce.

For the custom payment gateway you can set:

- Title
- Description
- Instructions
- Email instructions
- Icon
- Enable for shipping methods
- Accept for virtual orders
- Default order status
- Send additional emails
- Custom return URL (Thank You page)
- Minimum order amount ([Pro version](https://wpfactory.com/item/custom-payment-gateways-woocommerce/) only)

== Installation ==

1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start by visiting plugin settings at "WooCommerce > Settings > Custom Payment Gateways".
